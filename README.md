# Coding Challenge 07 - Small Business Landing Page

## Description

This project is a responsive landing page for a small business built using HTML, CSS, and JavaScript. It demonstrates modern web design principles, responsive layouts, and dynamic content to create an engaging user experience.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## Features

- Responsive page layout
- Modern HTML and CSS styling
- Interactive JavaScript functionality
- Call-to-action section
- Clean and organized design

## What I Learned

This project helped me improve my front-end web development skills by combining HTML, CSS, and JavaScript to build a responsive and visually appealing website. It also reinforced the importance of organizing code and creating user-friendly interfaces.
